No code chunks are showing in the poster.
No output warnings or messages are showing.
Figures and Tables Table 1 summarizing participant characteristics is present and overall.
If purposeful model selection: Table/figure with regression results is present.
If LASSO: Table/figure with variable importance is present.
At least one additional helpful figure (e.g., DAG, AUC-ROC, model fit, unadjusted relationship) is included.
All tables and figures have titles/captions.
Tables and figures do NOT have variable names as they appear in the data frame.
Variable names in figures/tables are understood by a reader (full words).
Figures and tables appear on the same page or close to where they are first referenced.
Tables and figures are an appropriate size and readable.
Figures and tables are clear and crisp (not blurred).
Colors in figures are only used if conveying information.
Intended message of figures is easily understood (e.g., ordered x-axis).
Introduction Provides good background for the research question.
Includes motivation for the research question.
References previous research that justifies the analysis (for purposeful model selection).
Clearly states the guiding question (and mentions prediction if LASSO).
Section is non-technical and broadly understandable.
Introduction has 5-8 bullet points.
Introduction contains references.
